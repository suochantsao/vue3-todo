# Online Todo List

It's a basic todolist website with member features.

# Features

1. Vue 3
2. Vue Router
3. axios

# Installation

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files

```
npm run lint
```
